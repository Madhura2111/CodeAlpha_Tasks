#  Image Gallery Web


## About the Project

This project is a responsive image gallery web application developed using **HTML, CSS, and JavaScript**.
It displays images in a structured grid layout and allows users to interact with them through a clean and modern interface.
Users can click on any image to open it in a **lightbox view**, where they can navigate between images using **Next** and **Previous** buttons.
The design adapts to different screen sizes, making it suitable for desktops, tablets, and mobile devices.

---

## Features

✔ Responsive design for all screen sizes  
✔ Lightbox preview mode  
✔ Next and Previous navigation  
✔ Image filtering by category  
✔ Smooth hover animations  

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript
