# 🎵 Music Player Web App

A simple and responsive **Music Player** built using **HTML, CSS, and JavaScript**.
This project demonstrates core frontend skills with interactive audio controls and a clean UI.

---

## 🚀 Features

* ▶ Play / Pause functionality
* ⏭ Next / Previous song
* 📃 Display song title and artist
* ⏱ Show current duration
* 🔊 Volume control
* 🔁 Autoplay next song

---

## 📁 Project Structure

```
project/
│
├── index.html
├── style.css
├── script.js
│
└── music/
      ├── song1.mp3
      ├── song2.mp3
      └── song3.mp3
```

---

## ⚙️ How to Run

1. Download or clone the project
2. Make sure you have a **music folder** with `.mp3` files
3. Open `music.html` in your browser

---

## 🎧 How to Add Songs

1. Add your `.mp3` files inside the `songs` folder
2. Open `music.js`
3. Update the songs array:

```javascript
const songs = [
{
title: "Song Name",
artist: "Artist Name",
src: "songs/song1.mp3"
}
];
```
