# Basic Calculator

A simple and responsive web-based calculator built using HTML, CSS, and JavaScript.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Real-time result preview
- Clear (C) button
- Delete (DEL) button
- Keyboard support
- Responsive and modern UI design
---
## Technologies Used

- HTML5
- CSS3 (Grid Layout)
- Vanilla JavaScript (DOM Manipulation)
---
## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click on `calculator.html`.
4. The calculator will open in your browser.
---
## Keyboard Support

- Numbers (0-9)
- Operators (+ - * /)
- Enter → Calculate result
- Backspace → Delete last character
- Escape → Clear all
---
## How It Works

- User input is stored as a string expression.
- Button clicks and keyboard inputs update the expression.
- The result is calculated using JS.
- Real-time preview updates automatically while typing.
